NexBoard

A customizable personal dashboard that brings weather, tasks, news, and a whiteboard into one interface.

Features

  • Weather Widget (Open-Meteo)
  • Todo List
  • News Feed (Google News RSS)
  • Whiteboard
  • Custom Layout

Tech Stack

  • React 18 + Vite
  • React Router v6
  • Open-Meteo
  • Google News RSS
  • react-grid-layout
  • localStorage
  • Netlify
